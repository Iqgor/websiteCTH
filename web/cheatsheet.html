Cheatsheet van map naar Github

Met commandpromt werken.

1. Maak een map in de locatie die je wil.
2. Open cmd
3. Typ in cd (locatie map)
4. Al zit je in het folder typ git init dit maakt een repositorie aan
5. Add alle bestanden die door “git add .” te doen.
5. Add alle bestanden die door “git add .” te doen.
6. Maak daarna je eerste commit aan door “git commit -m “beschrijving”


Repositorie maken en pushen naar github.

7.Maak dan een nieuwe repositorie aan in GitHub.
8.Volg dan de stappen die in de repositorie staan in GitHub, Typ deze dan in je cmd. (git remote add origin https://github.com/Iqgor/m.git git branch -M main git push -u origin main)
9. Als het goed is heb je nu je nieuwe repositorie in GitHub aan gemaakt.
